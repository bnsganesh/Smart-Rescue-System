Our Work

We had created a Dataset in which all types of Gestures and Objects will be included and Trained. which helps Jetson-Nano Dev Kit and Python Libraries to recognize and detect various Hand Gestures, Signs , objects and persons. We had UAVs instead of Helicopters and Land-Vehicles for transport purpose. And we had attached FPV(Camera), GSM and GPS modules along with above mentioned Kit and UAV, which detects Real-time objects and Tracks location then sends location along with visuals. So that, the Rescue team will go to the location and rescues the Tracked or Recognized people or object. And they'll also have the live Feed, therefore they can use understood. SRS also supplies some essential items like medicine to the Victim before Rescue team reaches them.
